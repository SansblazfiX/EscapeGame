PLAN D'OPÉRATION

0 / 4 missions complétées

💡 Comment jouer ?
1. Rends-toi physiquement dans la salle indiquée.
2. Scanne le QR code affiché sur place pour ouvrir l'énigme.
3. Résous l'énigme pour obtenir un chiffre secret.
4. Une fois les 4 chiffres collectés → entre le code final !
🔒 Code final — résous d'abord toutes les énigmes ← Retour à l'accueil
